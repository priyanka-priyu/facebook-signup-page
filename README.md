 Facebook Signup Page (HTML & CSS)

This project is a simple recreation of the Facebook signup page using HTML and CSS.  
It is built for practice and to showcase frontend design skills.

 Features
- Responsive layout using HTML & CSS
- Styled form elements (input fields, buttons)
- Clean design inspired by Facebook’s signup page

 Project Structure
- `index.html` → Main signup page
- `style.css` → Stylesheet for layout and design

 Purpose
This project is for learning and practicing frontend development.
It demonstrates form design, CSS styling, and layout techniques.
